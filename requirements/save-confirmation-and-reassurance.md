# Save Confirmation & Emotional Reassurance

## Context & Problem
When parents are tired or distracted, they may be unsure whether a moment has been successfully captured. This uncertainty reduces trust in the product.

## Goals
- Provide clear reassurance that a memory has been saved  
- Increase user confidence and emotional safety  

## Non-Goals
- No notifications or reminders  
- No sharing or follow-up prompts  
- No gamification  

## Functional Requirements
- After a recording is saved, the system provides immediate confirmation  
- Confirmation is clearly visible and unambiguous  
- Feedback uses calm, reassuring language or visuals  

## Constraints
- Confirmation must not interrupt the user flow  
- Must work consistently across devices  

## Success Criteria
- Reduced user uncertainty about saved recordings  
- Low number of support questions related to missing entries
